IMAGE ASSETS
============
Place PNG/WEBP images here. Paths must match the imageAsset fields in
GameConfig (Driver / RiderItem / TransformationForm).

Expected files:
  driver_mighty.png             - Mighty Driver banner image
  gashat_mighty_action_x.png    - Mighty Action X Gashat thumbnail
  gashat_taddle_quest.png       - Taddle Quest Gashat thumbnail
  exaid_level2.png              - Ex-Aid Level 2 full artwork
  exaid_level5.png              - Ex-Aid Level 5 full artwork

Recommended sizes:
  Driver banner:  400 x 280 px
  Item thumbnail: 128 x 128 px
  Form artwork:   512 x 768 px  (portrait)
