3D MODEL ASSETS
===============
Place Sceneform (.sfb) or glTF/GLB (.glb) model files here.
File names must match the asset paths defined in GameConfig.Models / TransformationForm.

Expected files:
  exaid_base.sfb          - Ex-Aid base form model
  exaid_level2.sfb        - Ex-Aid Action Gamer Level 2
  exaid_level5.sfb        - Ex-Aid Wizard Gamer Level 5
  gashat_mighty.sfb       - Mighty Action X Gashat item
  gashat_taddle.sfb       - Taddle Quest Gashat item
  mighty_driver.sfb       - Mighty Driver belt model

How to create .sfb files:
  - Use Sceneform tools plugin (Android Studio plugin) to convert FBX/OBJ/glTF.
  - Or use GLB files directly (set setIsFilamentGltf(true) in ModelLoader).

Free 3D model sources:
  - Sketchfab (check licence)
  - TurboSquid (check licence)
